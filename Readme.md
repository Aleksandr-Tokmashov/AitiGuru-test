# Тестовое задание AitiGuru
## Инструкция по установке
- Клонируйте репозиторий: \
`git clone git@github.com:Aleksandr-Tokmashov/AitiGuru-test.git` \
`cd AitiGuru-test`
- Запустите приложение: \
`sencha app watch`
- Откройте приложение в браузере по ссылке: \
`http://localhost:1841/`
